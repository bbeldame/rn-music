# Music Composer

## Features

[![Demo](https://raw.githubusercontent.com/bbeldame/rn-music/master/demo.png)](https://raw.githubusercontent.com/bbeldame/rn-music/master/demo.png)

## Installation 

```
npm install

react-native run-android
```

Or more simply: 

```
adb install -r musiccomposer.apk
```
